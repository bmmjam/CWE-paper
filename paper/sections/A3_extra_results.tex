\subsection{Pairwise significance (weak policy)}
Paired McNemar on leaf accuracy, seed 13, $n{=}300$ (consistent across seeds):
$\mctsreas(I{=}8)$ vs.\ $\mctstax(I{=}16)$: $57/24$ discordant pairs,
$p = 3.2\times10^{-4}$;
$\mctsreas(I{=}2)$ vs.\ $\mctstax(I{=}16)$ (at \emph{half} the budget): $59/23$,
$p = 8.7\times10^{-5}$;
$\mctsreas(I{=}2)$ vs.\ greedy: $80/19$, $p = 4.4\times10^{-10}$;
$\mctstax(I{=}16)$ vs.\ greedy: $p = 8\times10^{-6}$;
$\mctstax(I{=}16)$ vs.\ self-consistency ($N{=}8$): $p = 2\times10^{-3}$.
The full strategy $\times$ strategy matrix with Bonferroni adjustment is part of
the final protocol.

\subsection{Taxonomy-search saturation probe}
On the full-depth MITRE tree (716 leaves), $\mctstax$ at $I \in \{16, 64, 128\}$
returned byte-identical predictions on a small dev probe ($n{=}8$; $13$k, $38$k,
and $71$k tokens/example respectively): once the level-wise priors lock in, more
iterations re-visit the same branch. This motivated capping $\mctstax$ at
$I{=}16$ in the pilot grid.

\subsection{Planned ablations (final protocol)}
The following are pre-registered but not yet run at pilot scale:
exploration-constant sensitivity ($c_{\text{exp}} \in \{0.5, 1.0, 1.41\}$);
aggregation rule (majority vs.\ reward-weighted vs.\ robust-child);
tree-granularity breakdown (coarse vs.\ full-depth); per-language stratification
(the pilot is C/C++ only); wall-clock latency tables (P50/P95) separating
parallelizable from sequential token spend.
